Residential Management System

Full-Stack Web Application (Private Repository)

Ongoing Project

Overview

A full-stack web application designed to support residential and property management operations, including incident reporting, maintenance tracking, and role-based access for staff and residents. The system focuses on real-world workflows, data integrity, and usability for non-technical users.

Note: The source code is currently private due to active development and data sensitivity. Code access is available upon request.

Problem Statement

Property and residential management teams often rely on:

  • Emails and spreadsheets
  • Manual tracking of incidents and maintenance
  • Poor visibility of issue status and accountability

This leads to:

  • Delayed responses
  • Lost information
  • Inefficient communication between teams and residents

Solution

I designed and built a centralised web application that:

  • Tracks incidents and maintenance requests end-to-end
  • Provides role-based access for staff and residents
  • Improves transparency, accountability, and response times
  • Replaces fragmented manual processes with structured workflows
